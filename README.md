# QA Automation Experiments

This repository contains personal experiments and practice projects in QA automation. 

The goal of this repository is to explore and learn:

- Automation techniques across UI and API testing
- Integration with CI/CD pipelines
- Framework design and reusable test utilities
- Best practices for scalable, maintainable automated testing

This repo is intended as a sandbox to prototype ideas, test new tools, and refine automation skills in real-world-like scenarios.
